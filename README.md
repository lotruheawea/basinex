# basinex

The basin extractor. Extract a basin for a given gauging station.


## Dependencies

- numpy v1.14.5 or later
- netCDF4
- GDAL
- pyyaml v5.1 or later


## Installation

    pip install basinex


## Documentation

The main input file `input.yml` is documented and should (hopefully) give an overview
